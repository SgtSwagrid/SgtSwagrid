## Hi there, I'm Alec. Welcome to my GitHub profile!

I'm a recent Master's graduate from [ETH Zürich](https://ethz.ch/en.html), where I specialised in [theoretical computer science](https://en.wikipedia.org/wiki/Theoretical_computer_science) and [machine learning](https://en.wikipedia.org/wiki/Machine_learning). I completed my [thesis](https://github.com/SgtSwagrid/microscopic-railway-scheduling) with the [Institute for Transport Planning and Systems](https://www.ivt.ethz.ch/en/), where I developed novel algorithms for automated railway scheduling. Before that, I was a software engineer at [Immersive Technologies](https://www.immersivetechnologies.com/) in my home city of [Perth](https://en.wikipedia.org/wiki/Perth) in Western Australia.

I'm interested and well-versed in the worlds of algorithms, math, and [functional](https://en.wikipedia.org/wiki/Functional_programming) programming. Conversely, I would like to learn more about electronics, physics, and economics; a day where nothing is learnt is a day wasted!
